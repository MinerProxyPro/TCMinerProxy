## TCMinerSystem

> [!TIP]
> TCMinerSystem может проксировать обычные майнинг-пулы, а также позволяет превратить ваше оборудование в полноценный узел майнинг-пула в зависимости от ваших потребностей.
> 
> Оба способа позволяют вашему оборудованию добывать различные криптовалюты. Если вы управляете майнинг-площадкой или владеете узлом, вы можете настроить собственные комиссионные ставки для увеличения дохода.
> 
> Кроме того, TCMinerSystem предоставляет сопутствующий локальный безопасный клиент.[RMS](https://github.com/mine-Proxy/RMS), [RMS](https://github.com/mine-Proxy/RMS)Способно шифровать и сжимать данные в 10–20 раз, что значительно снижает     нагрузку на пропускную способность сети и предотвращает атаки посредников, повышая общую безопасность. Даже при использовании SIM-карты 4G одна карта обеспечивает стабильную работу более 5000 майнинг-устройств.
> 
>Перед началом использования внимательно ознакомьтесь с нашим [Соглашением об обслуживании](https://tcminersystem.gitbook.io/tcminersystem/guan-yu/fu-wu-xie-yi) 。
> 
> Если вам необходимо подключить оборудование к обычным майнинг-пулам, ознакомьтесь с разделом [Прокси обычных майнинг-пулов](https://tcminersystem.gitbook.io/tcminersystem/chuan-tong-kuang-chi-dai-li/dai-li-chuan-tong-kuang-chi)              Руководство。
> 
> Если вы хотите создать собственный узел майнинг-пула, изучите раздел [Самостоятельный узел майнинг-пула](https://tcminersystem.gitbook.io/tcminersystem/chuan-tong-kuang-chi-dai-li/dai-li-chuan-tong-kuang-chi) Руководство。



## 安装TCMinerSystem

> TCMinerSystem предоставляет установочные файлы (бинарные исполняемые файлы) для различных платформ и архитектур. Развертывание в Linux выполняется одной командой, а в Windows достаточно скачать исполняемый файл, запустить его двойным кликом и следовать инструкциям для начала работы.

### WINDOWS

> 1. Откройте следующую ссылку:
> 2. https://github.com/mine-Proxy/TCMinerSystem/tree/main/windows
> 3. Найдите последнюю версию программы RustMinerSystem, дождитесь полной загрузки страницы после перехода и нажмите «View raw» для скачивания.
> 4. Запустите скачанную программу двойным кликом, затем перейдите по ссылке в браузере согласно подсказке для входа в панель управления TCMinerSystem.
>5. Стандартные логин и пароль панели управления: qzpm19kkx xloqslz913

### Linux
> - Рекомендуемая операционная система — Ubuntu
> - Скопируйте и выполните следующие команды shell для запуска пакета инструментов

```
bash <(curl -s -L https://github.com/mine-Proxy/TCMinerSystem/raw/main/install.sh) 
```

> - Если загрузка в некоторых регионах слишком медленная или доступ невозможен, попробуйте **альтернативный адрес для установки**.

```
bash <(curl -s -L https://proxy.tcminersystem.com/install.sh)
```

> - После успешного запуска появится следующее меню, установите согласно инструкции.

> ![screenshot](https://github.com/mine-Proxy/TCMinerSystem/raw/main/image/install.gif)
